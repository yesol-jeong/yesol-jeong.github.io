# yesol-jeong.github.io

Static personal website for GitHub Pages.

## Local preview

Open `index.html` directly in a browser, or run a small local server from the
repository root:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Assets to replace

- `assets/profile.jpg`: profile photo used on the page.
- `assets/cv.pdf`: CV linked from the resume section.
